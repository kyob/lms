LMS Userpanel theme "bclean"
==========
Features
--------
  - use Bootstrap v4
  - (almost) valid HTML5
  - clean and minimal template

Requirments
--------
  - Bootstrap v4
  - Tether library for Tooltips

### Version
0.2.0

### Install Bootstrap with composer (default)
Got to theme directory ```userpanel/style/bclean/```
```sh
composer install
// or
composer update
```
